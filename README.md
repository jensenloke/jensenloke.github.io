# Jensen Loke

Static personal site served with GitHub Pages, containing writing and conference talks.

## Local Preview

Open `index.html` in a browser, or serve the folder with any static file server.

## Adding Articles

1. Add a new HTML file under `articles/`.
2. Link the new article from `index.html`.
3. Reuse `assets/styles.css` for shared layout and article components.

## Adding Talks

1. Add a self-contained presentation under `talks/<talk-slug>/`.
2. Keep public-safe media inside that talk folder.
3. Link the presentation from the Talks section in `index.html`.

This site intentionally has no build step.
