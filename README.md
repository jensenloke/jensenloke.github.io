# Jensen Loke

Static personal blog served with GitHub Pages.

## Local Preview

Open `index.html` in a browser, or serve the folder with any static file server.

## Adding Articles

1. Add a new HTML file under `articles/`.
2. Link the new article from `index.html`.
3. Reuse `assets/styles.css` for shared layout and article components.

This site intentionally has no build step.
